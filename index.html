기능 2. 자동 제어

1번 축과 2번 축을 동시에 제어하며, 레시피 기반으로 상승/하강 위치와 속도 조건을 설정합니다. 반복 횟수는 10,000회 이상 자동 반복 설정이 가능합니다.

기능 3. 반복 동작 데이터 분석 평가표

반복 동작 이후 속도, 위치 편차, 딜레이 데이터를 분석하여 Excel 형식으로 제공합니다.

분석 결과
5회 반복 동작 모두 정상 범위 내에서 수행되었으며, 속도 변화, 가속 시간, 위치 편차, 왕복 딜레이 값이 안정적으로 유지되었습니다.
동작 시퀀스 설명
1. 초기화 — Servo ON → Home Move → Wait Home Complete
2. 자동 사이클 — Auto Cycle Start 후 Z축 상대위치 이동
3. 위치 확인 — Encoder Position Polling → Z1/Z2 InPosition Check
4. 반복 처리 — Delay → Move Z Down → 엔코더 피드백 → 횟수 증가
5. 종료 판정 — 반복 횟수 도달 시 Complete, 미도달 시 사이클 재시작
